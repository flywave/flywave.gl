# 为 Flywave.gl 贡献代码

感谢您对为 Flywave.gl 贡献代码的兴趣！我们欢迎社区的贡献，以帮助改进我们的 3D 地图渲染引擎。

## 如何贡献

### 报告问题

如果您发现 bug 或有功能请求，请在我们的 GitHub 仓库上[提交问题](https://github.com/flywave/flywave.gl/issues/new)。报告 bug 时，请包括：

- 清晰且具有描述性的标题
- 重现问题的步骤
- 预期行为
- 实际行为
- 适用的截图或代码示例
- 您的环境信息（操作系统、浏览器等）

### 提交拉取请求

1. Fork 仓库
2. 为您的功能或 bug 修复创建新分支
3. 进行修改
4. 如适用，请编写测试
5. 确保所有测试通过
6. 使用清晰的提交消息提交您的更改
7. 推送到您的 fork
8. 向主仓库提交拉取请求

### 代码风格

- 遵循项目中的现有代码风格
- 对所有新代码使用 TypeScript
- 编写清晰、描述性的注释
- 确保您的代码有良好的文档

## 开发环境设置

1. 克隆您 fork 的仓库
2. 使用 `pnpm install` 安装依赖
3. 使用 `pnpm test` 运行测试
4. 使用 `pnpm start` 启动开发服务器

## 行为准则

请注意，该项目发布了贡献者行为准则。参与此项目的人员即表示同意遵守其条款。

## 有疑问？

如果您对贡献有任何疑问，请随时在 GitHub 上[发起讨论](https://github.com/flywave/flywave.gl/discussions)。
