# @flywave/flywave-webtile-datasource

## Overview

This module provides functionality to access various web map tile services, including:
- [HERE Map Tile API 2.1](https://developer.here.com/documentation/map-tile/topics/quick-start-map-tile.html)
- OpenStreetMap
- Mapbox
- ArcGIS
- Bing Maps
- Custom tile services

These services can be used to obtain tiles of fixed size and use them as a base map, for example,
to display satellite images or street maps.
